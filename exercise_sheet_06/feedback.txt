Python Tips:
	You can swap items in Python by using a tuple:
	(a, b) = (b, a)

	The brackets can be omitted:
	a, b = b, a

	This would reduce the swap operation to:
	self._list[index_1], self._list[index_2] =\
		self._list[index_2], self._list[index_1]

	self._list[index_1].set_heap_index(index_1)
	self._list[index_2].set_heap_index(index_2)
